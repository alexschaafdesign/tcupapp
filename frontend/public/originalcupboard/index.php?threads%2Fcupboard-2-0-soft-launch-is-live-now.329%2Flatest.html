edit: Cupboard 2.0 is live! go to portal.tcupboard.org to sign up (you'll have to make a new account there, but its VERY QUICK i promise )

It's in 'beta' phase, which is a dumb phrase but basically there are probably still lots of little kinks to iron out, but it'll be easier to find them once more people start using it, so have at it! I'm leaving this all up for a good while, so you can always reply to this thread with any problems.




edit: still going to try and have it up March 5, but we're (thankfully) finding some bugs now, so wanna wait til its more robust.

ahoy!

So i've been working on moving the Cupboard to a new custom-built platform, which will give us a better look and feel, more options for adding different sections, and i think it will just be COOLER and more FUN - and the plan is to shift over it to next week on the 5th.

it should have most of the same features as this current Cupboard, and will eventually grow to have LOTS more (band profiles being a big one)

the only aspect that matters to you is that starting this week, going to tcupboard.org will redirect to the new site, and you'll have to make a new account there (will take literally 5 seconds), and reupload an avatar, etc. But i'll eventually preserve a lot of the "Cupboard 1.0" chats as 'historical threads' or something, so we won't be losing any key info that's already here. (and this site will still exist for a while, just under a different domain)

if you're feeling curious, you can make an account over there right now, by going to portal.tcupboard.org (this would actually be helpful to me, if you sort of 'test' it out, and let me know any problems or bugs, before the full switch next week)

one other small note, I changed ever so slightly the way that the chat/threads will work -- when you create a post, you can assign various 'tags' to it, like "General chat" "Looking for bandmates", etc -- and then when viewing the chat, you can filter by those tags to find the different categories of posts, rather than having discrete 'rooms' for each category. hopefully this feels pretty straightforward and intuitive.
